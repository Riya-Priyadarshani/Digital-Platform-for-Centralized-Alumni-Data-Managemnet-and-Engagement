# Digital Platform for Centralized Alumni Data Management and Engagement

## Overview

The Digital Platform for Centralized Alumni Data Management and Engagement is designed to streamline the process of maintaining alumni records and fostering meaningful interaction between alumni and institutions. The platform provides a unified system to store, manage, and update alumni information while enabling communication, networking, and engagement.

## Features

* Centralized database for alumni records
* User authentication and secure access
* Alumni profile creation and management
* Search and filter functionality
* Communication tools for announcements and updates
* Event management and participation tracking
* Admin dashboard for data management and analytics

## Objectives

* Simplify alumni data management
* Improve communication between alumni and institutions
* Enhance alumni engagement and participation
* Provide data insights for institutional growth

## Technology Stack

* Frontend: HTML, CSS, JavaScript and ejs
* Backend: Node.js with Express
* Database: MySQL 
* Version Control: Git

## Installation

1. Clone the repository: https://github.com/Riya-Priyadarshani/Digital-Platform-for-Centralized-Alumni-Data-Managemnet-and-Engagement

   ```bash
   git clone https://github.com/your-username/Digital-Platform-for-Centralized-Alumni-Data-Managemnet-and-Engagement.git
   ```
2. Navigate to the project directory:

   ```bash
   cd Digital-Platform-for-Centralized-Alumni-Data-Managemnet-and-Engagement
   ```
3. Install dependencies:

   ```bash
   npm install
   ```
4. Run the application:

   ```bash
   npm start or npm run dev
   ```

## Usage

* Register as an alumni or admin
* Create and update profiles
* Browse alumni directory
* Participate in events and discussions
* Receive updates and announcements
